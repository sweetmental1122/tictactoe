# TicTacToe

## Setup

### 1. MySQL
Create the database (tables are auto-created on server start):
```sql
CREATE DATABASE tictactoe;
```

### 2. Backend
```bash
cd backend
cp .env.example .env   # fill in your DB credentials
npm install
npm run dev
```

### 3. Frontend
```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173
